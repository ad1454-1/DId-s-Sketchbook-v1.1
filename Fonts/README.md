# Fonts 目录说明

这里存放的是“输出到底图上的文字”可选字体，不是程序界面字体。

当前已加入：

- `ZCOOLQingKeHuangYou-Regular.ttf`
  - 字体名：站酷庆科黄油体
  - 风格：标题感、海报感、较强装饰性
- `ZCOOLKuaiLe-Regular.ttf`
  - 字体名：站酷快乐体
  - 风格：活泼、圆润、偏可爱
- `MaShanZheng-Regular.ttf`
  - 字体名：马善政
  - 风格：手写感、行笔感更强

这些字体用于高级配置里的“输出字体文件”下拉菜单，控制生成到底图中的文字样式。

字体来源：

- Google Fonts / google/fonts

许可证：

- SIL Open Font License 1.1

建议：

- 界面字体继续优先使用 `Microsoft YaHei` 这类可读性更高的系统字体
- 艺术字体更适合用于输出文字，而不是程序界面
